一个基于PHP开发的捐赠管理系统，提供捐赠记录管理、用户登录等功能。

## ✨ 功能特点

- 捐赠记录管理
- 用户登录/登出
- 捐赠数据可视化展示
- 多支付渠道支持(微信、支付宝)

- PHP 7.0+

2. 配置数据库连接(`db_connection.php`)
3. 访问`index.html`开始使用
